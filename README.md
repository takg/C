# C Programming Examples

This repository contains a collection of **C programs** demonstrating core concepts of the C programming language.  
It is intended for **learning, practice, interviews, and reference**.

The repository includes programs related to:

  - Data types and sizes
    - [sizeof_datatype.c](https://github.com/takg/C/blob/main/src/sizeof_datatype.c)
    - [limits_datatype.c](https://github.com/takg/C/blob/main/src/limits_datatype.c)
  - Input / Output
- Control Structures
  - if / switch
  - loops (for, while, do-while)
- Functions
  - Call by value / reference
  - Recursion
- Arrays & Strings
- Pointers
  - Pointer arithmetic
  - Pointers and arrays
- Structures & Unions
- Memory Management
  - `malloc`, `calloc`, `realloc`, `free`
- File Handling
- Bit Manipulation
- Miscellaneous Utilities

> Each program is written to be **simple, readable, and standards-compliant**.
